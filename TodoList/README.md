# TODO Console Application

This is a simple console application built with C# that allows users to manage a list of TODOs. The application lets you add, remove, and view your TODOs through a command-line interface.

## Features
- **Add a TODO**: Enter a unique, non-empty TODO item.
- **Remove a TODO**: Select and remove a TODO by its index from the list.
- **View all TODOs**: Displays all current TODOs in a numbered list.
- **Exit**: Exits the application.
